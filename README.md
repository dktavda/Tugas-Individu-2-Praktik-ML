## Hasil Praktik Dasar Python
Berikut adalah spesifikasi lingkungan kerja dan poin materi yang telah dipraktikkan:
- **Editor:** Visual Studio Code (VS Code)
- **Bahasa Pemrograman:** Python 3.11
- **Topik Utama:** Input/Output, Variabel, Casting Data, Operator (Aritmatika & Logika), dan Eksplorasi Library.

### Langkah-Langkah Praktik di VS Code:
1. **Setup File:** Membuat file python dengan ekstensi `.py` (atau Jupyter Notebook `.ipynb`) di dalam VS Code.
2. **Implementasi I/O:** Mempraktikkan fungsi `print()` dengan berbagai teknik formatting dan fungsi `input()` untuk mengambil input data user melalui Terminal VS Code.
3. **Eksperimen Variabel:** Memahami sifat *Dynamic Typing* Python dengan mengamati perubahan tipe data variabel secara langsung.
4. **Casting Variable:** Melakukan konversi tipe data (integer, float, string) menggunakan fungsi built-in.
5. **Operasi Matematis & Logika:** Menjalankan operasi aritmatika dasar dan logika perbandingan untuk pengolahan data.
6. **Eksplorasi Dokumentasi:** Menggunakan fungsi `dir()` dan `help()` di terminal terintegrasi VS Code untuk melihat fungsi-fungsi dalam library `math` dan `time`.

## Analisis & Kesimpulan
Berdasarkan praktikum menggunakan VS Code ini:
- **VS Code Efficiency:** Penggunaan VS Code sangat memudahkan proses debugging dan eksekusi kode melalui *Integrated Terminal*. 
- **Type Casting:** Saya menyimpulkan bahwa fungsi `input()` di Python selalu menghasilkan *String*. Maka, casting ke *Integer* atau *Float* sangat penting sebelum melakukan kalkulasi numerik.
- **Kunci Computer Vision:** Dasar-dasar variabel dan operator ini adalah langkah awal yang krusial sebelum memanipulasi matriks gambar menggunakan library OpenCV atau NumPy nantinya.
